# vrc_facial_lock_generator

# 概要

VRChat向けのアバターに表情固定ギミックを自動生成するEditor拡張です。

bd_さまの作成した[ModularAvatar](https://modular-avatar.nadena.dev/ja/)を利用してセットアップするため、元々のFxレイヤーを書き換えることなく利用できます。  
不要になった際も1つGameObjectを削除するだけで完了できます。
